# @urolov/react-native-xprinter

Package for controlling xprinters

## Installation


```sh
npm install @urolov/react-native-xprinter
```


## Usage


```js
import { multiply } from '@urolov/react-native-xprinter';

// ...

const result = multiply(3, 7);
```
